# React hooks

React 官方提供了很多 hooks，https://react.dev/reference/react

- useCallback
- useContext
- useDebugValue
- useDeferredValue
- useEffect
- useId
- useImperativeHandle
- useInsertionEffect
- useLayoutEffect
- useMemo
- useReducer
- useRef
- useState
- useSyncExternalStore
- useTransition

其中常见的是 useEffect useState useRef 这几个 hooks
